# wikidict-wordlist - Wikipedia Monolingual Reference Data (wordlists)

This repository makes available a collection of wordlists derived from article titles in various language Wikipedias. The data has been extracted from [Wikidata](https://www.wikidata.org/).

## Data

The data directory contains subdirectories arranged in order of [ISO language code](http://en.wikipedia.org/wiki/ISO_639-1).

The basic filename pattern is `[ISO]-wordlist_wiki.txt`, with `[ISO]` being the target language ISO code. A list of all available languages is [below](#available-language-pairs).
